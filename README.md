# 1024house-deploy
1024House 是一款本地部署、数据完全私有的情侣互动工具，支持 Windows / Mac / NAS / Docker 一键运行，所有数据存储在用户自己设备，不上云、不泄露、无惧平台跑路与数据丢失。 面向情侣与年轻家庭，提供时光记录、纪念日管理、礼账人情、愿望清单、简记收支、私密聊天、宝宝成长与疫苗记录、勋章成就等一体化功能，采用粉色系圆角卡片设计，移动优先适配，界面简洁易用。 前后端纯自研，无第三方框架依赖，基于 Node.js + Express + MongoDB 构建，支持离线使用、文件上传、数据批量导入导出，情侣数据共享、宝宝数据隔离，兼顾隐私安全与使用体验。 适合追求私密、可控、长久可用的双人生活管理，一次部署永久使用，真正属于自己的爱情与家庭数字小屋。
